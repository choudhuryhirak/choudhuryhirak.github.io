The Kepler Observatory

Actionscript Application


This project is based on NASA's Kepler Probe.It is a flash application built using Actionscript 3.0 to present scientific data in an interactive format.The App aims to present a wide range of Information both scientific such as Exoplanet data and infomercial, eg: Posters and Promotions of Kepler in a consolidated format. It was a hectic task to sort out volumes of data into a compact format of an application. I was really glad when it was done

This application was created exclusively in the Adobe Flash environment(Adobe Flash CS6).

The Application interface consists of the following segments:

  • An Overview Page with a custom planet gallery and Videos backlink
  • A Discovery Page of Kepler satellites
  • The Kepler Education and Research Program
  • The Kepler Launch information and statistics etc
  • Video gallery viewable within the application.
  • Poster gallery viewable within the application.

This is a project for the module "Scripting for Interactivity" completed for MSc Multimedia(Interaction Design and computing) course by the University of Westminster.
