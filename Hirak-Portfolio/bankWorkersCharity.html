Bank Worker's Charity

UX Analysis and Project Management


This project deals with the re-design of Bank Worker's charity website for the module Multimedia Management.

Aim

Our Aim was to create a new redesign of the existing website of Bank Worker's charity which follows proper segmentation of information and offers a clean navigation to the users.

Process

The process for the redesign included a review and critical analysis of the existing website.A new Information Architecture was established with user-flows and use-case scenarios.A functional prototype was developed within a proposed schedule and budget.

Highlights of the New Design

The new design followed a problem-oriented approach to issues compared to the solution-driven approach of the existing website.The exisiting website has the following limitations:

  • Solution Oriented
  • Overwhelming with too much information.
  • Unclear User Journey
  • Difficult to find relevant information fast.

The idea for the new website was to solve these issues with a design approach which was:

  • Problem Oriented
  • Clean design with only relevant information on set topics
  • Clear, with user journeys not exceeding more than three steps
